# OpenAudit Hub Demo Dataset

This public demo uses an aggregated scan snapshot for the public website `truckhardware.com.au`.

Included in the demo:

- 3 Lighthouse page summaries
- 20 Pa11y page checks
- 279 grouped accessibility findings
- example issue categories, affected page patterns, and remediation steps

Not included:

- raw Lighthouse HTML/JSON reports
- raw Pa11y evidence files
- databases
- credentials
- private Docker configuration
- customer or internal data

# OpenAudit Hub 演示数据

这个公开演示页面使用公开网站 `truckhardware.com.au` 的汇总扫描快照。

演示中包含：

- 3 个 Lighthouse 页面摘要
- 20 个 Pa11y 页面检查摘要
- 279 个分组后的无障碍问题
- 示例问题类别、受影响页面模式和修复建议

不包含：

- 原始 Lighthouse HTML/JSON 报告
- 原始 Pa11y 证据文件
- 数据库
- 账号密码
- 私有 Docker 配置
- 客户或内部数据
